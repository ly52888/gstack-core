# Brand Guidelines

## Relationship To Codex And OpenAI

gstack-core is designed for Codex workflows, but it is an independent open-source project.

Allowed positioning:

- Codex-first workflow router
- built for Codex skills
- works with Codex plugins, skills, MCP tools, CLI commands, and local scripts
- for builders using Codex

Avoid:

- official Codex
- OpenAI gstack
- Codex by OpenAI
- OpenAI-certified
- OpenAI-backed
- using OpenAI logos, wordmarks, or trade dress without permission

OpenAI's brand page states that OpenAI names, logos, ChatGPT/GPT brands, and other OpenAI trademarks are OpenAI property. Keep naming and visuals independent.

Source: https://openai.com/brand/

## Naming

Recommended public name:

```text
gstack-core
```

Recommended subtitle:

```text
Codex-first intelligent workflow router
```

Recommended disclaimer:

```text
Independent open-source project. Not affiliated with, endorsed by, or sponsored by OpenAI.
```

## Voice

- Practical
- Evidence-first
- Local-first
- Developer-facing
- Governance-aware

Avoid vague hype like "fully autonomous AI employee" or "one-click build anything."
