# Closeout

Use this reference before saying work is complete.

## Required Closeout

- State what changed.
- State what was verified and with which command/tool.
- State what could not be verified.
- State changed file paths.
- Clean temporary files unless useful deliverables.
- Update project docs/handoff when present.

## Do Not Claim Done If

- Tests were not run and no reason is given.
- Browser/visual work was not inspected.
- Generated media was not reviewed.
- GitHub/account action was only planned but not executed.
- Context handoff is needed but missing.

## Route Trace

For non-trivial work, append or report:

```text
Route:
Why this route:
Tools used:
Fallbacks:
Verification:
Remaining risk:
```
