# Product Positioning

## One-Liner

gstack-core is a Codex-first intelligent skill router that turns plain-language work into the right skill, plugin, tool, verification path, and handoff.

Use `Codex-first` and `for Codex workflows` as compatibility positioning. Do not imply that gstack-core is an official OpenAI or Codex product.

## Short Pitch

AI coding agents fail in predictable ways: they waste context, ask for information they can inspect, pick the wrong tool, start coding before requirements are stable, and claim completion without evidence. gstack-core provides a single workflow entry point that routes tasks, asks only necessary questions, uses local evidence first, and closes work with verification.

Alternative short pitch:

> gstack-core gives Codex a workflow brain: route the task, gather evidence, ask only what matters, pick the right tool, and verify before saying done.

## Primary Value

- Less token waste from repeated context and broad file reads.
- Fewer wrong turns through route preview and capability cards.
- Better starts through multi-AI council and evidence-first planning.
- Better finishes through closeout, docs, cleanup, and route trace.
- Safer tool use through clarification gates, external AI disclosure rules, and fallback policy.

## Target Users

- Solo builders using Codex for real projects.
- Small teams that want repeatable AI-assisted development workflows.
- Agencies building websites/apps from reference sites and client briefs.
- Operators who need creative, GitHub, browser QA, and code work from one entry point.
- Teams evaluating internal agent governance.

## Differentiation

- Codex-first skill structure rather than a generic prompt pack.
- Progressive disclosure: small main skill, loaded references, capability cards.
- Natural command routing for page bugs, screenshots, failed tasks, QA, docs, and closeout.
- Multi-round external AI council for pre-build planning and legally safer inspired redesign.
- Bundled local codegraph for orientation and impact checks.
- Open core with a clear path to commercial governance and workflow packs.

## Not The Pitch

Do not position gstack-core as:

- A fully autonomous software engineer.
- A legal/compliance substitute.
- A SaaS platform today.
- A replacement for project-specific tests, reviews, or production controls.
- A scraper or website cloning tool.
- An official OpenAI, ChatGPT, or Codex product.

## Taglines

- Stop prompting. Start routing.
- One skill to route the work.
- Give Codex a workflow brain.
- From vague request to verified closeout.
- Evidence first, tools second, completion last.
- The missing router layer for Codex workflows.
